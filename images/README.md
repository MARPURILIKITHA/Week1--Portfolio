# Images Directory

This directory contains all image assets for the portfolio website.

## Required Images

- `profile.jpg` - Your profile picture (recommended size: 400x400px or larger, square aspect ratio)
- `project1.jpg` - Project screenshots (recommended size: 800x600px or larger)

## Image Guidelines

1. **Profile Picture**: Use a professional headshot or portrait photo
2. **Project Images**: Use high-quality screenshots of your projects
3. **Optimization**: Compress images before uploading to improve page load speed
4. **Formats**: Use JPG for photos, PNG for graphics with transparency

## Placeholder Images

If you don't have images yet, you can:
- Use placeholder services like [Placeholder.com](https://placeholder.com/)
- Use [Unsplash](https://unsplash.com/) for free stock photos
- Create your own images using design tools
